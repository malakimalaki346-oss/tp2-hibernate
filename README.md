TP 2 : Entités Salle et Utilisateur avec Hibernate

Étape 1 : Création du projet

J’ai commencé par créer un projet Maven dans IntelliJ IDEA et configurer les dépendances nécessaires, notamment Hibernate, H2 Database et JUnit. J’ai ensuite structuré le projet selon l’architecture standard Maven afin de faciliter le développement et les tests.

Étape 2 : Développement et configuration

J’ai implémenté les entités, la configuration Hibernate ainsi que les classes principales permettant la connexion à la base de données H2. J’ai aussi ajouté des tests unitaires avec JUnit pour vérifier le bon fonctionnement de l’application.

Étape 3 : Exécution et validation

Enfin, j’ai exécuté l’application et les tests pour valider le fonctionnement du projet. Après vérification, le projet a été versionné et préparé pour être déposé sur GitHub avec un fichier README descriptif.
